任务使命

行星探路者实验室专注于地外探测车系统的前沿研究。 我们开发能够在月球与火星表面自主行驶、避障和规划路线的智能探测车。

融合物理学、电子工程、计算机科学多学科知识， 为本科生提供真实的科研实践平台。

CORE TASKS

  • 🌙 月球车自主导航系统研发
  • 🔴 火星车环境感知与路径规划
  • 🛰️ 探测载荷集成与野外科学观测
