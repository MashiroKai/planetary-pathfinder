任务使命

行星探路者实验室面向未来月球基地与火星基地开发做准备。 在月球及火星表面生存条件恶劣，多数情况不能由人直接进行工作，具有智能的机器人则变得尤为关键。

月球拥有丰富宝贵资源：天然真空环境可供天文观测、丰富的氦 3 资源、南极永久冰原区的水资源。 火星作为人类第二居住星球，是人类成为多行星物种的第一步。

CORE TASKS

  • 🌙 月球车自主导航系统研发
  • 🔴 火星车环境感知与路径规划
  • 🛰️ 探测载荷集成与野外科学观测
